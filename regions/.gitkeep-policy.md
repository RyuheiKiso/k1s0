# .gitkeep ポリシー

regions/ 配下の `.gitkeep` ファイルは CLI のスキャフォールド機能に必要なディレクトリ構造を維持する目的で配置されている。

## 削除してはいけない理由

CLI の `scan_existing_dirs()` 関数（`CLI/crates/k1s0-cli/src/commands/generate/steps.rs`）が
`regions/business/` および `regions/service/` 配下のディレクトリ構造をスキャンし、
新規コンポーネント生成時の選択肢として提示する。

`.gitkeep` を削除すると Git がディレクトリを追跡しなくなり、
クローン後の CLI 実行時にスキャフォールドが正しく動作しない。

## 対象

- `regions/business/*/server/{go,rust}/`
- `regions/business/*/library/{go,rust,ts,dart}/`
- `regions/business/*/database/`
- `regions/business/*/client/flutter/`
- `regions/service/*/server/{go,rust}/`
- `regions/service/*/database/`
- `regions/system/server/go/`
- `regions/system/library/{go,rust,typescript,dart}/`
