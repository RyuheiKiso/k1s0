# セットアップツール

開発環境のセットアップ作業を自動化するためのツールです。  
CLI（コマンドライン）とGUI（グラフィカル画面）の両方から利用できます。

---

## モジュール構成

| モジュール | 役割 | 技術スタック |
|-----------|------|------------|
| `cli`     | コマンドラインインターフェース | Rust |
| `gui`     | グラフィカルインターフェース | Tauri（Rust） |
| `common`  | CLI・GUI 共通ロジック | Rust |

`common` モジュールに共通処理をまとめ、`cli` と `gui` の両方から参照する構成です。

---

## ディレクトリ構成

```
setup/
├── cli/        # CLIツール
├── gui/        # GUIアプリ（Tauri）
├── common/     # CLI・GUI共通ロジック
└── README.md   # このファイル
```

---

## 利用可能なコマンド（随時追加予定）

| コマンド | 説明 |
|---------|------|
| `install-check` | 必要なソフトウェア（Node.js、Rust、Go など）がインストールされているか確認する |
| `help` | コマンド一覧と使い方を表示する |
