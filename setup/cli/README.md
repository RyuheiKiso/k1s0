# cli

コマンドラインから開発環境のセットアップを行うツールです。

---

## 技術スタック

| 項目 | 内容 |
|------|------|
| 言語 | Rust Edition 2024 |
| 共通ロジック | `common` クレート |

---

## ディレクトリ構成

```
cli/
├── src/
│   └── main.rs     # エントリーポイント
└── Cargo.toml      # パッケージ設定
```

---

## ビルド・実行

```bash
# ビルド
cargo build --release

# 実行
./target/release/k1s0 <コマンド>
```

---

## 利用可能なコマンド（随時追加予定）

| コマンド | 説明 |
|---------|------|
| `install-check` | 必要なソフトウェア（Node.js、Rust、Go など）がインストールされているか確認する |
| `help` | コマンド一覧と使い方を表示する |
