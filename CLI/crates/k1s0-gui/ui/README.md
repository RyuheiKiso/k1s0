# k1s0 GUI UI

Frontend for the Tauri-based `k1s0` desktop application.

## Commands

```bash
npm ci
npm run dev
npm run test
npm run build
```

## Notes

- This package lives under `CLI/crates/k1s0-gui/ui`.
- CI for the desktop app should run Node commands from this directory.
- Generated desktop bundles are produced by the parent Tauri crate in `CLI/crates/k1s0-gui`.
