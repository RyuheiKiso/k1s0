# Frontend Flutter Template

Flutter フロントエンドのテンプレート。

## ステータス

置き場のみ固定。実装は後続フェーズで行う。

## feature/ の生成物（予定）

```
feature/frontend/flutter/{name}/
├── pubspec.yaml
├── README.md
├── config/
│   ├── default.yaml
│   ├── dev.yaml
│   ├── stg.yaml
│   └── prod.yaml
├── lib/
│   ├── main.dart
│   └── src/
│       ├── presentation/
│       ├── application/
│       ├── domain/
│       └── infrastructure/
└── deploy/
    ├── base/
    └── overlays/
```
