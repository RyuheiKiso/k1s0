# Frontend React Template

React フロントエンドのテンプレート。

## ステータス

置き場のみ固定。実装は後続フェーズで行う。

## feature/ の生成物（予定）

```
feature/frontend/react/{name}/
├── package.json
├── README.md
├── config/
│   ├── default.yaml
│   ├── dev.yaml
│   ├── stg.yaml
│   └── prod.yaml
├── src/
│   ├── presentation/
│   ├── application/
│   ├── domain/
│   ├── infrastructure/
│   └── main.tsx
└── deploy/
    ├── base/
    └── overlays/
```
