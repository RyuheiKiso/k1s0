## 本ディレクトリが採番する IMP-DIR ID

本ディレクトリは方針定義そのものであり、直接の IMP-DIR 採番はしない。各方針が適用される具体配置は 10_ルートレイアウト以降の章で IMP-DIR-\* を採番する。
