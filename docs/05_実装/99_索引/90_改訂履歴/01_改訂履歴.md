# 99. 索引 / 90. 改訂履歴 / 01. 改訂履歴

本ファイルは `05_実装/` 全 12 章の IMP-\* ID の採番・変更・廃止履歴を時系列で集約する。IMP-TRACE-POL-007（改訂履歴の本章集約）により、各章への分散記録は禁止されており、横断的な傾向分析・四半期レビュー・年次監査・半期 Technology Radar 更新の一次参照先となる。

## 改訂履歴の記載項目

各エントリは以下を含める（IMP-TRACE-POL-007 に基づく）。

- 改訂日 / 改訂 PR 番号（リリース時点 初版は PR 番号なし）
- 対象 IMP-\* ID（範囲表記可）
- 変更種別（Add / Modify / Supersede / Deprecate）
- 変更理由（ADR リンクまたは節ファイルへのリンク）
- 影響範囲（該当する他 IMP-\* / ADR / NFR）
