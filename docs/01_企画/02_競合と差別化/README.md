# 02_競合と差別化

04 章が「個別 OSS」の話なのに対し、本章は「商用 IDP / k8s ディストリビューション / Dapr / Cloud Foundry などのプラットフォーム製品」に対する k1s0 の立ち位置を示す章。目的は、決裁者が「商用を買った方が早いのでは」という問いに対し、定性比較（差別化）と定量比較（TCO / Build vs Buy）の両面で回答を持てるようにすること。

## 想定読者

経営層・稟議決定者が第一読者。営業・企画職（顧客提案で競合対比を使う者）が第二読者。本章は短くまとまっており、30 分で通読できる。
