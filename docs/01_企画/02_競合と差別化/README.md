# 02_競合と差別化

本章は商用 IDP / k8s ディストリビューション / Dapr / Cloud Foundry などのプラットフォーム製品に対する k1s0 の立ち位置を示す。目的は、k1s0 を採用するか商用製品を採用するかを判断する際に、「商用を買った方が早いのでは」という問いに対して定性比較（差別化）と質的なリスク構造比較（Build vs Buy）の両面で回答を持てるようにすることである。

## 想定読者

採用判断者（情シス管理職・エンジニアリングマネージャ）が第一読者。営業・企画職（顧客提案で競合対比を使う者）が第二読者。本章は短くまとまっており、30 分で通読できる。
