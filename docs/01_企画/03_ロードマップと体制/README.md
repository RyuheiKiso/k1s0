# 03_ロードマップと体制

k1s0 を「Phase 0（企画承認）→ Phase 5（全社ロールアウト）」の 5 年計画として分解し、各フェーズの期間・人員・ハードウェア・成果物・判断ゲートを定義する章。MVP-0（起案者単独・VM 1 台・デモ）から段階的に拡張する設計で、「一度に全部作らない」ことを前提にする。体制面ではアーキテクト / infra / tier1 / 運用の役割をフェーズ別に割り当てる。

本章は併せて **進行判断ゲート（Go/No-Go 基準・KPI 閾値・撤退トリガー）** も所管する（[`03_KPIと承認基準.md`](./03_KPIと承認基準.md)）。「続けるか畳むか」の意思決定は実行計画の一部であり、金額試算（[`../04_定量試算/`](../04_定量試算/)）とは目的軸が異なるため、本章側に配置する。

## 想定読者

経営層（全社投資計画との整合確認）、プロジェクト責任者（マイルストーン設計）、参画メンバー（自分の役割と期待水準の把握）、PMO（KPI 監視と Phase 遷移判定会議の運営）。
