# 01_背景と目的

k1s0 を「なぜ作るのか」に答える章。現状の JTC 情シス部門が抱える構造的課題と、k1s0 が成立した場合に提供される価値を定義し、後続章（アーキテクチャ・技術選定・投資試算）で共有される判断軸を本章で固定する。本章で「痛み」と「勝ち筋」が読者の合意を取れなければ、以降の技術選定や投資判断は机上の空論として崩れる。

## 想定読者

第一読者は決裁者（情シス管理職・役員）。第二読者は同じ痛みを共有する情シス実務者。技術者視点の深掘りは [`../../02_構想設計/01_アーキテクチャ/`](../../02_構想設計/01_アーキテクチャ/) 以降に委ねる。本章は「金の話」ではなく、「今この基盤を作らなかったときに何を失うか」を語る。

## 前章と後章

- 前章: なし（本書で最初に読む章）。エグゼクティブサマリだけなら [`../企画書.md`](../企画書.md) を先に読むとよい。
- 後章: [`../../02_構想設計/01_アーキテクチャ/`](../../02_構想設計/01_アーキテクチャ/) — 本章で定義した痛みに対する構造的回答。
