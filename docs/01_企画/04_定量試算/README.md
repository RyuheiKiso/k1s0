# 04_定量試算

[`03_ロードマップと体制/`](../03_ロードマップと体制/) で定義した工数・人員・期間を、実際の金額に換算する章。試算前提（企業規模 3 段階・人月単価・インフラ単価）を先に公開し、そのうえで Build（k1s0 内製）vs Buy（商用 IDP）の 5 年 TCO を比較する。数字を動かせば結論が変わる可能性があるため、試算前提の出典（JIAS / 経産省 / レバテック等）を明記している。

本章は **企画全体における「金額の唯一の一次ソース」** である。他章（例: [`../02_競合と差別化/03_BuildVsBuy.md`](../02_競合と差別化/03_BuildVsBuy.md)）は本章を参照するのみで数値を重複保持しない。Go/No-Go 基準・段階承認ライン・撤退判断トリガーは [`../03_ロードマップと体制/03_KPIと承認基準.md`](../03_ロードマップと体制/03_KPIと承認基準.md) に集約した（金額そのものではなく「進行判断」の位置付けのため）。

## 想定読者

経営層・CFO・稟議決定者（金の判断）。事業責任者（年次予算との整合）。Phase 進行中は PMO が Go/No-Go 判定に [`../03_ロードマップと体制/03_KPIと承認基準.md`](../03_ロードマップと体制/03_KPIと承認基準.md) を参照する。
