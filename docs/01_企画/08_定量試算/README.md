# 08_定量試算

07 章で定義した工数・人員・期間を、実際の金額に換算する章。試算前提（企業規模 3 段階・人月単価・インフラ単価）を先に公開し、そのうえで Build（k1s0 内製）vs Buy（商用 IDP）の 5 年 TCO を比較する。数字を動かせば結論が変わる可能性があるため、試算前提の出典（JIAS / 経産省 / レバテック等）を明記している。あわせて Go/No-Go 基準と撤退判断トリガーも本章で定義する。

## 想定読者

経営層・CFO・稟議決定者（金の判断）。事業責任者（年次予算との整合）。Phase 進行中は PMO が Go/No-Go 判定に [`04_KPIと承認基準.md`](./04_KPIと承認基準.md) を参照する。

## 前章と後章

- 前章: [`../07_ロードマップと体制/`](../07_ロードマップと体制/) — 工数・人員・期間を受け取る。
- 後章: [`../09_法務とコンプライアンス/`](../09_法務とコンプライアンス/) — 金が合う前提でリスクを確認する。
