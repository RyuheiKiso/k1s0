# Frontend Framework

k1s0 Frontend Framework は、フロントエンド開発のための共通パッケージ群を提供します。個別機能チームが「画面の中身」以外（ナビゲーション/レイアウト/デザイン/権限制御/設定読込/観測）を再実装せずに済む状態を実現します。
