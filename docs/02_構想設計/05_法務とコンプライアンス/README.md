# 05_法務とコンプライアンス

採用 40+ OSS のライセンス義務を「利用形態別」に整理し、AGPL-3.0（Grafana / Loki / MinIO / Renovate 等 6 個）を「社内利用では義務が発動しない」と FSF FAQ・各プロジェクト FAQ を出典として構造的に説明する章。併せて k1s0 自身の知財戦略（OSS 化するか、クローズドのままにするか）を整理する。申請における「AGPL は使えないのでは」という最終懸念への回答書として機能する。

## 想定読者

法務部門・内部監査・コンプライアンス担当（定期ライセンス監査の根拠）、経営層・申請決定者（最後のリスク確認）、OSS ポリシーを持つ情シス部門。
