# ヒアリングシート

## 目的

3 案の意思決定 ([`05_3案の深掘り評価.md`](./05_3案の深掘り評価.md) 8 節) と PoC スコープ確定 ([`06_PoC計画.md`](./06_PoC計画.md)) のために、ユーザー側から取得すべき情報を構造化した質問票。本シートは PoC キックオフ前の 1〜2 回のヒアリング (各 90 分) で消化することを想定している。

回答が「不明 / 未定」となった項目は **PoC で検証する対象** に格上げするか、**仮置き値で進める旨を明記** して進める。空欄のままにしない。

回答記入欄は質問の右側 (`回答:` 行) を埋める運用。本シートは Markdown のまま編集し、PoC 開始時点でリポジトリにコミットする。

## 10. 確認結果の取り扱い

ヒアリング結果を本シートに記入後、以下の流れで PoC スコープを確定する。

1. PM がヒアリング結果を集約 → アーキテクトが [`05_3案の深掘り評価.md`](./05_3案の深掘り評価.md) の TCO・スコア表を再計算。
2. 案 A / B / C の推奨を再算定し、ステアリングで方針合意。
3. 合意内容を [`06_PoC計画.md`](./06_PoC計画.md) のスコープ・期間・予算に反映。
4. 確定値を [`07_定量モデル.md`](./07_定量モデル.md) の各モデル変数に投入。
5. 本シートはリポジトリにコミットし、PoC 期間中も追記更新する。
