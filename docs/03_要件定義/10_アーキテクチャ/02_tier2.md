# ARC-T2: tier2 要件

tier2 ドメインサービスの責務、tier1 クライアントライブラリ利用規約、他 tier2 との連携ルールを定義する。tier2 からは tier1 の内部言語は不可視であり、公開 gRPC / クライアントライブラリのみを経由する。
