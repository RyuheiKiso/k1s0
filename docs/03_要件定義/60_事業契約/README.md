# 60_事業契約

本カテゴリは k1s0 プロジェクトの非技術的な要件を集約する。技術的に実装可能であっても、法務・経営・調達・人事・CSR の合意が取れなければプロジェクトは進まない。本カテゴリはこれら非技術要件を明文化し、組織横断で合意するための文書群である。

## 本カテゴリの狙い

JTC の稟議では、技術要件よりも「いくらかかるのか」「ライセンス違反は起きないか」「教育はどうするのか」「テナント追加のコストは誰が負担するのか」「生成 AI を使うならガバナンスはどうするのか」に多くの質問が集中する。本カテゴリはこれらに事前に答えを用意し、稟議時の Q&A で詰まる時間を最小化する。

## 読み順

完全通読は 01 から 13 までの順。部分参照では以下を目安にする。

- **経営層向け**: 05、06、10、12（コストと意思決定）
- **法務部向け**: 07、08、11（契約と責任）
- **調達部門向け**: 08、09（契約とサプライチェーン）
- **CSR・ESG 部門向け**: 13（サステナビリティ）
- **tier2/tier3 リード向け**: 03、04（オンボーディングと教育）
