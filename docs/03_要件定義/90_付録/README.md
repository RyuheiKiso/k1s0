# 90_付録

本カテゴリは、k1s0 要件定義書の本編からは独立した参照資料を集める。共通フレーム 2013 との対応表、非機能要求グレードの判定、外部参考文献など、要件そのものではないが要件の根拠を補強する資料を収録する。

## 本カテゴリの位置付け

要件本編を読み進める中で「なぜこの粒度で分解されたのか」「この数値の根拠は何か」「どの標準を参照したのか」という疑問が必ず生じる。付録はその疑問に即座に答えるための検索可能な参照集として機能する。申請・外部監査・新規参加者の教育で本編だけでは示せない根拠を補うのが主目的。

## ファイル構成

- [01_共通フレーム2013対応表.md](01_共通フレーム2013対応表.md): IPA 共通フレーム 2013 のプロセスステップと k1s0 要件の対応
- [02_非機能要求グレード判定.md](02_非機能要求グレード判定.md): IPA 非機能要求グレード 2018 の社会的影響度モデル判定と各カテゴリのレベル
- [03_参考文献.md](03_参考文献.md): 外部参考文献（IPA 文献、CNCF 仕様、法令、業界標準、書籍、論文）

## 運用ルール

付録は要件の変更に連動して更新する必要がある。参照先の文献が更新（法改正、標準改訂、CNCF 卒業/アーカイブ等）された際は四半期ごとのレビューで反映。対応表は要件 ID が増減した際に PR で同時更新する。
