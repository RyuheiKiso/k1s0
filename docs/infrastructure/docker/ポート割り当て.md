# ポート割り当て一覧

Docker Compose で定義されている全サービスのポート割り当て。全ホストポートは `${VAR:-default}` 形式で環境変数化されており、`.env` で変更可能。

## インフラサービス

| サービス | REST/HTTP | gRPC | 用途 |
|----------|-----------|------|------|
| postgres | 5432 | - | PostgreSQL データベース |
| mysql | 3306 | - | MySQL データベース |
| redis | 6379 | - | Redis キャッシュ・分散ロック |
| redis-session | 6380 | - | Redis（BFF セッション管理用） |
| kafka | 9092 | - | Kafka メッセージブローカー |
| kafka-ui | 8090 | - | Kafka 管理 UI |
| schema-registry | 8081 | - | Avro/JSON Schema レジストリ |
| keycloak | 8180 / 9000(管理) | - | 認証・認可基盤（IdP） |
| vault | 8200 | - | シークレット管理 |
