# service tier クライアント開発

## 概要

service tier のクライアントは、BFF が提供する GraphQL / REST API を通じてエンドユーザーに業務機能を届ける。Web は React、モバイル/デスクトップは Flutter で構築する。

いずれのクライアントも system tier の client SDK と business tier の共通コンポーネントに依存する。

```
service client → business client（領域共通 UI）→ system client SDK（認証・API 基盤）
```

### TanStack Query（サーバー状態管理）

BFF の GraphQL / REST API からのデータ取得・キャッシュ・再取得を管理する。

```typescript
import { useQuery, useMutation, useQueryClient } from "@tanstack/react-query";
import { api } from "@/shared/lib/api";

// クエリキーの定義
export const taskKeys = {
  all: ["tasks"] as const,
  lists: () => [...taskKeys.all, "list"] as const,
  detail: (id: string) => [...taskKeys.all, "detail", id] as const,
};

// データ取得
export function useTask(id: string) {
  return useQuery({
    queryKey: taskKeys.detail(id),
    queryFn: () => api.get<Task>(`/api/v1/tasks/${id}`),
  });
}

// ミューテーション
export function useCreateTask() {
  const queryClient = useQueryClient();
  return useMutation({
    mutationFn: (data: CreateTaskInput) =>
      api.post<Task>("/api/v1/tasks", data),
    onSuccess: () => {
      queryClient.invalidateQueries({ queryKey: taskKeys.lists() });
    },
  });
}
```

### Zustand（クライアント状態管理）

サーバーに保存しない UI 状態（フィルター、選択状態等）を管理する。

```typescript
import { create } from "zustand";

interface TaskFilterStore {
  statusFilter: string | null;
  searchQuery: string;
  setStatusFilter: (status: string | null) => void;
  setSearchQuery: (query: string) => void;
}

export const useTaskFilterStore = create<TaskFilterStore>((set) => ({
  statusFilter: null,
  searchQuery: "",
  setStatusFilter: (status) => set({ statusFilter: status }),
  setSearchQuery: (query) => set({ searchQuery: query }),
}));
```

### TanStack Router（ルーティング）

型安全なルーティングを提供する。認証ガードは system client SDK が提供するものを利用する。

```typescript
import { createFileRoute } from "@tanstack/react-router";
import { requireAuth } from "system-client/auth";

export const Route = createFileRoute("/tasks/$taskId")({
  beforeLoad: requireAuth,  // system client SDK の認証ガード
  component: TaskDetailPage,
  loader: async ({ params }) => {
    return queryClient.ensureQueryData({
      queryKey: taskKeys.detail(params.taskId),
      queryFn: () => api.get<Task>(`/api/v1/tasks/${params.taskId}`),
    });
  },
});
```

### React Hook Form + Zod（フォーム・バリデーション）

```typescript
import { useForm } from "react-hook-form";
import { zodResolver } from "@hookform/resolvers/zod";
import { z } from "zod";

const createTaskSchema = z.object({
  customerId: z.string().uuid(),
  items: z.array(
    z.object({
      productId: z.string().uuid(),
      quantity: z.number().int().positive(),
    })
  ).min(1, "1件以上の商品が必要です"),
});

type CreateTaskInput = z.infer<typeof createTaskSchema>;

function CreateTaskForm() {
  const { register, handleSubmit, formState: { errors } } = useForm<CreateTaskInput>({
    resolver: zodResolver(createTaskSchema),
  });
  const createTask = useCreateTask();

  return (
    <form onSubmit={handleSubmit((data) => createTask.mutate(data))}>
      {/* フォームフィールド */}
    </form>
  );
}
```

### Riverpod（状態管理）

```dart
import 'package:riverpod_annotation/riverpod_annotation.dart';

part 'task_providers.g.dart';

// 非同期データ取得
@riverpod
Future<Task> task(TaskRef ref, String id) async {
  final repository = ref.watch(taskRepositoryProvider);
  return repository.getTask(id);
}

// 一覧取得
@riverpod
Future<List<Task>> taskList(TaskListRef ref) async {
  final repository = ref.watch(taskRepositoryProvider);
  return repository.getTasks();
}

// ミューテーション
@riverpod
class TaskNotifier extends _$TaskNotifier {
  @override
  Future<Task?> build() async => null;

  Future<void> createTask(CreateTaskInput input) async {
    state = const AsyncLoading();
    state = await AsyncValue.guard(() async {
      final repository = ref.read(taskRepositoryProvider);
      return repository.createTask(input);
    });
  }
}
```

### go_router（ルーティング）

```dart
import 'package:go_router/go_router.dart';
import 'package:system_client/auth.dart';

final appRouter = GoRouter(
```

### freezed（モデル定義）

```dart
class Task with _$Task {
```

### dio（HTTP クライアント）

```dart
import 'package:dio/dio.dart';
import 'package:system_client/api.dart';

class TaskRepository {
  final Dio _dio;

  TaskRepository(this._dio);

  Future<Task> getTask(String id) async {
    final response = await _dio.get('/api/v1/tasks/$id');
    return Task.fromJson(response.data);
  }

  Future<Task> createTask(CreateTaskInput input) async {
    final response = await _dio.post(
      '/api/v1/tasks',
      data: input.toJson(),
    );
    return Task.fromJson(response.data);
  }
}

// dio インスタンスは system_client SDK が提供するインターセプターを適用
final dio = createDio()
  ..interceptors.addAll([
    AuthInterceptor(ref.read(authProvider)),       // system_client: 認証トークン付与
    TelemetryInterceptor(),                        // system_client: トレース伝播
  ]);
```
