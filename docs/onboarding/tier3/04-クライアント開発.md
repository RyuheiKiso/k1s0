# service tier クライアント開発

## 概要

service tier のクライアントは、BFF が提供する GraphQL / REST API を通じてエンドユーザーに業務機能を届ける。Web は React、モバイル/デスクトップは Flutter で構築する。

いずれのクライアントも system tier の client SDK と business tier の共通コンポーネントに依存する。

```
service client → business client（領域共通 UI）→ system client SDK（認証・API 基盤）
```

### TanStack Query（サーバー状態管理）

BFF の GraphQL / REST API からのデータ取得・キャッシュ・再取得を管理する。

```typescript
import { useQuery, useMutation, useQueryClient } from "@tanstack/react-query";
import { api } from "@/shared/lib/api";

// クエリキーの定義
export const orderKeys = {
  all: ["orders"] as const,
  lists: () => [...orderKeys.all, "list"] as const,
  detail: (id: string) => [...orderKeys.all, "detail", id] as const,
};

// データ取得
export function useOrder(id: string) {
  return useQuery({
    queryKey: orderKeys.detail(id),
    queryFn: () => api.get<Order>(`/api/v1/orders/${id}`),
  });
}

// ミューテーション
export function useCreateOrder() {
  const queryClient = useQueryClient();
  return useMutation({
    mutationFn: (data: CreateOrderInput) =>
      api.post<Order>("/api/v1/orders", data),
    onSuccess: () => {
      queryClient.invalidateQueries({ queryKey: orderKeys.lists() });
    },
  });
}
```

### Zustand（クライアント状態管理）

サーバーに保存しない UI 状態（フィルター、選択状態等）を管理する。

```typescript
import { create } from "zustand";

interface OrderFilterStore {
  statusFilter: string | null;
  searchQuery: string;
  setStatusFilter: (status: string | null) => void;
  setSearchQuery: (query: string) => void;
}

export const useOrderFilterStore = create<OrderFilterStore>((set) => ({
  statusFilter: null,
  searchQuery: "",
  setStatusFilter: (status) => set({ statusFilter: status }),
  setSearchQuery: (query) => set({ searchQuery: query }),
}));
```

### TanStack Router（ルーティング）

型安全なルーティングを提供する。認証ガードは system client SDK が提供するものを利用する。

```typescript
import { createFileRoute } from "@tanstack/react-router";
import { requireAuth } from "system-client/auth";

export const Route = createFileRoute("/orders/$orderId")({
  beforeLoad: requireAuth,  // system client SDK の認証ガード
  component: OrderDetailPage,
  loader: async ({ params }) => {
    return queryClient.ensureQueryData({
      queryKey: orderKeys.detail(params.orderId),
      queryFn: () => api.get<Order>(`/api/v1/orders/${params.orderId}`),
    });
  },
});
```

### React Hook Form + Zod（フォーム・バリデーション）

```typescript
import { useForm } from "react-hook-form";
import { zodResolver } from "@hookform/resolvers/zod";
import { z } from "zod";

const createOrderSchema = z.object({
  customerId: z.string().uuid(),
  items: z.array(
    z.object({
      productId: z.string().uuid(),
      quantity: z.number().int().positive(),
    })
  ).min(1, "1件以上の商品が必要です"),
});

type CreateOrderInput = z.infer<typeof createOrderSchema>;

function CreateOrderForm() {
  const { register, handleSubmit, formState: { errors } } = useForm<CreateOrderInput>({
    resolver: zodResolver(createOrderSchema),
  });
  const createOrder = useCreateOrder();

  return (
    <form onSubmit={handleSubmit((data) => createOrder.mutate(data))}>
      {/* フォームフィールド */}
    </form>
  );
}
```

### Riverpod（状態管理）

```dart
import 'package:riverpod_annotation/riverpod_annotation.dart';

part 'order_providers.g.dart';

// 非同期データ取得
@riverpod
Future<Order> order(OrderRef ref, String id) async {
  final repository = ref.watch(orderRepositoryProvider);
  return repository.getOrder(id);
}

// 一覧取得
@riverpod
Future<List<Order>> orderList(OrderListRef ref) async {
  final repository = ref.watch(orderRepositoryProvider);
  return repository.getOrders();
}

// ミューテーション
@riverpod
class OrderNotifier extends _$OrderNotifier {
  @override
  Future<Order?> build() async => null;

  Future<void> createOrder(CreateOrderInput input) async {
    state = const AsyncLoading();
    state = await AsyncValue.guard(() async {
      final repository = ref.read(orderRepositoryProvider);
      return repository.createOrder(input);
    });
  }
}
```

### go_router（ルーティング）

```dart
import 'package:go_router/go_router.dart';
import 'package:system_client/auth.dart';

final appRouter = GoRouter(
  redirect: (context, state) {
    // system_client SDK の認証ガード
    final isLoggedIn = ref.read(authStateProvider).isAuthenticated;
    if (!isLoggedIn) return '/login';
    return null;
  },

      builder: (context, state) => const OrderListScreen(),

  ],
);
```

### freezed（モデル定義）

```dart
import 'package:freezed_annotation/freezed_annotation.dart';

class Order with _$Order {
  const factory Order({
    required String id,

    required OrderStatus status,
```

### dio（HTTP クライアント）

```dart
import 'package:dio/dio.dart';
import 'package:system_client/api.dart';

class OrderRepository {
  final Dio _dio;

  OrderRepository(this._dio);

  Future<Order> getOrder(String id) async {
    final response = await _dio.get('/api/v1/orders/$id');
    return Order.fromJson(response.data);
  }

  Future<Order> createOrder(CreateOrderInput input) async {
    final response = await _dio.post(
      '/api/v1/orders',
      data: input.toJson(),
    );
    return Order.fromJson(response.data);
  }
}

// dio インスタンスは system_client SDK が提供するインターセプターを適用
final dio = createDio()
  ..interceptors.addAll([
    AuthInterceptor(ref.read(authProvider)),       // system_client: 認証トークン付与
    TelemetryInterceptor(),                        // system_client: トレース伝播
  ]);
```
