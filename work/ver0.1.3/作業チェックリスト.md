## 受け入れ基準

### 機能要件
- [x] `k1s0 doctor` で環境診断が実行される
- [x] 必須ツール（Rust, Node, pnpm）のチェックが行われる
- [x] オプションツール（Go, buf, Flutter等）のチェックが行われる
- [x] 問題発見時に推奨アクションが表示される
- [x] `--json` オプションでJSON出力される
- [x] `--verbose` オプションで詳細情報が表示される
- [x] `--check <category>` で特定カテゴリのみチェックできる

### 非機能要件
- [x] 1つのチェック失敗で全体が止まらない
- [x] Windows/macOS/Linux で動作する
- [x] `--no-color` オプションが機能する
- [x] 終了コードが適切に設定される
- [x] 全テストが通過する（149テスト）
