# k1s0 v0.1.7 作業チェックリスト

## 概要

ドメイン層の開発基盤（ドメインレジストリ）を構築する。

- **`k1s0 new-domain` コマンド強化**: 4言語対応のドメインスキャフォールドテンプレート
- **`k1s0 domain-catalog` コマンド新設**: ドメイン一覧と依存関係の可視化
- **`k1s0 domain-graph` コマンド新設**: 依存グラフの Mermaid / DOT 出力、循環依存検出

## 現状分析

### 実装済み（関連パッケージ）

- [x] k1s0-cli（clap ベースの CLI）
- [x] k1s0-generator（テンプレートエンジン + lint エンジン）
- [x] CLI/templates/（4種の feature テンプレート）
- [x] manifest.schema.json（ドメイン層フィールド定義済み）
- [x] K040-K047 lint ルール（ドメイン依存検証）
- [x] `k1s0 new-domain` コマンド（基本版）
- [x] `k1s0 domain-list` / `domain-version` / `domain-dependents` / `domain-impact` コマンド

### v0.1.7 で実装

- [x] ドメインテンプレート強化（Rust/Go/React/Flutter）
- [x] domain scanner モジュール
- [x] domain-catalog コマンド
- [x] domain-graph コマンド（Mermaid/DOT/循環検出）

### 実装方針の変更

チェックリスト当初は「新規テンプレートディレクトリ作成」と想定していたが、4言語のドメインテンプレートは **既に存在** していたため、**既存テンプレートの強化**（サンプルコード追加 + 条件付きイベント雛形）に変更した。

コマンド名は既存の flat 構造に合わせ `domain-catalog`, `domain-graph` として追加した（サブコマンドグループではなく独立コマンド）。
