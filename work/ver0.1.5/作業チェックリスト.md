# k1s0 v0.1.5 作業チェックリスト

## 概要

フロントエンドフレームワーク（React/Flutter）に Real-time 通信パッケージを追加する。

- **@k1s0/realtime**: React 向け WebSocket/SSE クライアント
- **k1s0_realtime**: Flutter 向け WebSocket/SSE クライアント

## 現状分析

### 実装済み（関連パッケージ）

- [x] @k1s0/api-client（HTTP クライアント）
- [x] @k1s0/auth-client（認証トークン管理）

- [x] k1s0_auth（Flutter 認証）

## 受け入れ基準

### 機能要件（WebSocket）

- [ ] WebSocket 接続・切断が正しく動作する
- [ ] メッセージの送受信が正しく動作する
- [ ] 自動再接続（Exponential backoff）が動作する
- [ ] ハートビート（ping/pong）が動作する
- [ ] 認証トークンが付与される
- [ ] 接続状態が正しく通知される

### 機能要件（SSE）

- [ ] SSE イベントストリームを受信できる
- [ ] イベントタイプ別のハンドラが動作する
- [ ] 自動再接続が動作する
- [ ] Last-Event-ID が送信される

### 機能要件（Realtime Provider/Manager）

- [ ] ネットワーク状態（online/offline）が検知される
- [ ] 複数の接続を一元管理できる
- [ ] オフラインキューにメッセージを蓄積できる
- [ ] オンライン復帰時にキューがフラッシュされる
- [ ] キューが永続化される

### 非機能要件

- [ ] TypeScript/Dart の型安全性が保たれている
- [ ] React 18/19 で動作する
- [ ] Flutter 3.16+ で動作する
- [ ] メモリリークがない（適切なクリーンアップ）
- [ ] 全テストが通過する
