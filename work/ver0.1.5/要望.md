# 要望

フロントエンドアプリケーションでリアルタイム通信機能を実装したい。
チャット、通知、ダッシュボードの自動更新など、サーバーからのプッシュ通知が必要なユースケースに対応したい。

## 回答（実装方針）

### 概要

1. **@k1s0/realtime**: React 向け Real-time 通信パッケージ
2. **k1s0_realtime**: Flutter 向け Real-time 通信パッケージ

### 技術選定

| フレームワーク | WebSocket | Server-Sent Events | 理由 |
|---------------|-----------|-------------------|------|
| **React** | ネイティブ WebSocket | EventSource API | 標準 API 活用、軽量 |
| **Flutter** | web_socket_channel | SSE パッケージ | 公式推奨パッケージ |

### サポートするプロトコル

| プロトコル | 方向 | ユースケース |
|-----------|------|-------------|
| **WebSocket** | 双方向 | チャット、ゲーム、リアルタイムコラボレーション |
| **Server-Sent Events (SSE)** | サーバー→クライアント | 通知、ダッシュボード更新、ログストリーミング |

## 機能 1: WebSocket クライアント

### React 版 (`@k1s0/realtime`)

```tsx
import { useWebSocket, WebSocketStatus } from '@k1s0/realtime';

interface ChatMessage {
  id: string;
  userId: string;
  content: string;
  timestamp: Date;
}

function ChatRoom({ roomId }: { roomId: string }) {
  const {
    status,
    lastMessage,
    sendMessage,
    connect,
    disconnect,
  } = useWebSocket<ChatMessage>({
    url: `wss://api.example.com/chat/rooms/${roomId}`,
    // 自動再接続設定
    reconnect: {
      enabled: true,
      maxAttempts: 5,
      backoff: 'exponential', // 1s, 2s, 4s, 8s, 16s
    },
    // 認証トークン付与
    protocols: [],
    headers: {
      Authorization: `Bearer ${token}`,
    },
    // ハートビート
    heartbeat: {
      enabled: true,
      interval: 30000, // 30秒
      message: JSON.stringify({ type: 'ping' }),
    },
    // イベントハンドラ
    onOpen: () => console.log('接続完了'),
    onClose: (event) => console.log('切断', event.code),
    onError: (error) => console.error('エラー', error),
    onMessage: (message) => {
      // メッセージ受信時の処理
      addMessageToList(message);
    },
  });

  const handleSend = (content: string) => {
    sendMessage({
      type: 'message',
      payload: { content },
    });
  };

  return (
    <div>
      <ConnectionStatus status={status} />
      <MessageList messages={messages} />
      <MessageInput onSend={handleSend} disabled={status !== 'connected'} />
    </div>
  );
}
```

### Flutter 版 (`k1s0_realtime`)

```dart
import 'package:k1s0_realtime/k1s0_realtime.dart';

class ChatNotifier extends StateNotifier<ChatState> {
  final K1s0WebSocket _socket;

  ChatNotifier(String roomId)
      : _socket = K1s0WebSocket(
          url: 'wss://api.example.com/chat/rooms/$roomId',
          reconnectConfig: ReconnectConfig(
            enabled: true,
            maxAttempts: 5,
            backoffType: BackoffType.exponential,
          ),
          heartbeatConfig: HeartbeatConfig(
            enabled: true,
            interval: Duration(seconds: 30),
            message: '{"type": "ping"}',
          ),
        ),
        super(ChatState.initial()) {
    _socket.messages.listen(_onMessage);
    _socket.status.listen(_onStatusChange);
  }

  void sendMessage(String content) {
    _socket.send({
      'type': 'message',
      'payload': {'content': content},
    });
  }

  void _onMessage(Map<String, dynamic> data) {
    final message = ChatMessage.fromJson(data);
    state = state.copyWith(messages: [...state.messages, message]);
  }

  void _onStatusChange(ConnectionStatus status) {
    state = state.copyWith(status: status);
  }

  @override
  void dispose() {
    _socket.close();
    super.dispose();
  }
}

// Riverpod Provider
final chatProvider = StateNotifierProvider.family<ChatNotifier, ChatState, String>(
  (ref, roomId) => ChatNotifier(roomId),
);
```

## 機能 2: Server-Sent Events (SSE) クライアント

### React 版

```tsx
import { useSSE } from '@k1s0/realtime';

interface Notification {
  id: string;
  title: string;
  body: string;
  type: 'info' | 'warning' | 'error';
  createdAt: Date;
}

function NotificationListener() {
  const { status, lastEvent, error } = useSSE<Notification>({
    url: '/api/notifications/stream',
    // 認証ヘッダー
    withCredentials: true,
    // イベントタイプ別ハンドラ
    eventHandlers: {
      notification: (data) => showNotification(data),
      alert: (data) => showAlert(data),
    },
    // 再接続設定
    reconnect: {
      enabled: true,
      maxAttempts: Infinity,
      interval: 3000,
    },
    onError: (error) => console.error('SSE エラー:', error),
  });

  return (
    <NotificationBell
      count={unreadCount}
      connected={status === 'connected'}
    />
  );
}
```

### Flutter 版

```dart
import 'package:k1s0_realtime/k1s0_realtime.dart';

final notificationStreamProvider = StreamProvider<Notification>((ref) {
  final sse = K1s0SSE(
    url: '/api/notifications/stream',
    reconnectConfig: ReconnectConfig(
      enabled: true,
      maxAttempts: null, // 無限
      interval: Duration(seconds: 3),
    ),
  );

  return sse.events
      .where((event) => event.eventType == 'notification')
      .map((event) => Notification.fromJson(event.data));
});

class NotificationBell extends ConsumerWidget {
  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final notifications = ref.watch(notificationStreamProvider);

    return notifications.when(
      data: (notification) => Badge(count: unreadCount),
      loading: () => CircularProgressIndicator(),
      error: (err, stack) => Icon(Icons.error),
    );
  }
}
```

## 機能 3: 接続状態管理とオフライン対応

### React 版

```tsx
import {
  RealtimeProvider,
  useConnectionStatus,
  useOfflineQueue
} from '@k1s0/realtime';

function App() {
  return (
    <RealtimeProvider
      config={{
        // オフライン時のメッセージキューイング
        offlineQueue: {
          enabled: true,
          maxSize: 100,
          persistToStorage: true, // localStorage に永続化
        },
        // ネットワーク状態監視
        networkMonitor: {
          enabled: true,
          onOnline: () => console.log('オンライン復帰'),
          onOffline: () => console.log('オフライン'),
        },
      }}
    >
      <MyApp />
    </RealtimeProvider>
  );
}

function ConnectionIndicator() {
  const { isOnline, isConnected, reconnecting } = useConnectionStatus();

  if (reconnecting) return <Chip label="再接続中..." color="warning" />;
  if (!isOnline) return <Chip label="オフライン" color="error" />;
  if (!isConnected) return <Chip label="切断" color="error" />;
  return <Chip label="接続中" color="success" />;
}

function ChatInput() {
  const { queue, flush, clear } = useOfflineQueue();
  const { sendMessage, status } = useWebSocket(...);

  const handleSend = (content: string) => {
    if (status === 'connected') {
      sendMessage({ content });
    } else {
      // オフライン時はキューに追加
      queue({ type: 'message', payload: { content } });
    }
  };

  // オンライン復帰時にキューをフラッシュ
  useEffect(() => {
    if (status === 'connected') {
      flush((item) => sendMessage(item));
    }
  }, [status]);

  return <Input onSend={handleSend} />;
}
```

### Flutter 版

```dart
import 'package:k1s0_realtime/k1s0_realtime.dart';

// アプリ全体で使用する接続マネージャー
final realtimeManagerProvider = Provider<K1s0RealtimeManager>((ref) {
  return K1s0RealtimeManager(
    offlineQueueConfig: OfflineQueueConfig(
      enabled: true,
      maxSize: 100,
      persistToStorage: true,
    ),
  );
});

// 接続状態プロバイダー
final connectionStatusProvider = StreamProvider<ConnectionInfo>((ref) {
  final manager = ref.watch(realtimeManagerProvider);
  return manager.connectionStatus;
});

class ConnectionIndicator extends ConsumerWidget {
  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final status = ref.watch(connectionStatusProvider);

    return status.when(
      data: (info) {
        if (info.reconnecting) {
          return Chip(label: Text('再接続中...'), backgroundColor: Colors.orange);
        }
        if (!info.isOnline) {
          return Chip(label: Text('オフライン'), backgroundColor: Colors.red);
        }
        if (!info.isConnected) {
          return Chip(label: Text('切断'), backgroundColor: Colors.red);
        }
        return Chip(label: Text('接続中'), backgroundColor: Colors.green);
      },
      loading: () => CircularProgressIndicator(),
      error: (_, __) => Icon(Icons.error),
    );
  }
}
```
