    ```rust
    #[arg(short = 'i', long)]
    pub interactive: bool,
    ```

  - [ ] 引数を `Option` に変更
    ```rust
    #[arg(short = 't', long = "type", value_enum)]

    #[arg(short, long)]
    pub name: Option<String>,  // 必須 → Option
    ```

  - [ ] `execute()` 関数の改修
    - 引数が不足している場合、prompts モジュールで補完

  - [ ] 対話フロー実装

    ```
    3. domain が未指定 + 既存domain存在 → ドメイン選択プロンプト
    4. オプション未指定 → オプション選択プロンプト
    5. 生成実行
    ```

## 受け入れ基準

### 機能要件
- [ ] 引数なしでコマンド実行時、対話モードが起動する
- [ ] `--interactive` / `-i` フラグで強制的に対話モードが起動する
- [ ] 一部引数指定時、不足分のみ対話で補完される
- [ ] 従来の引数指定方式が引き続き動作する

### 非機能要件
- [ ] CI環境（非TTY）で適切にエラー終了する
- [ ] `--no-color` オプションが対話モードでも機能する
- [ ] Ctrl+C で適切に中断できる
- [ ] 全テストが通過する
