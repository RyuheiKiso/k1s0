# 要望

フロントエンド開発の効率を向上させたい。
特にビジネスアプリケーションで頻繁に使用するデータテーブルとフォームの実装コストを削減したい。

## 回答（実装方針）

### 概要

1. **DataTable コンポーネント**: MUI DataGrid をベースにした高機能テーブル
2. **Form Generator**: MUI コンポーネント + Zod スキーマからフォームを自動生成

### 技術選定

| フレームワーク | DataTable | Form | 理由 |
|---------------|-----------|------|------|
| **React** | @mui/x-data-grid | MUI + react-hook-form + Zod | MUIエコシステム統合、型安全 |
| **Flutter** | カスタム実装 | カスタム実装 | Material 3 準拠、Riverpod統合 |

## 機能 1: DataTable コンポーネント

### React 版 (`@k1s0/ui`) - MUI DataGrid ベース

```tsx
import { K1s0DataTable, createColumns, dateColumn, actionsColumn } from '@k1s0/ui';

interface User {
  id: string;
  name: string;
  email: string;
  role: 'admin' | 'user' | 'guest';
  createdAt: Date;
}

// カラム定義ヘルパーを使用
const columns = createColumns<User>([
  { field: 'name', headerName: '氏名', flex: 1, sortable: true },
  { field: 'email', headerName: 'メール', flex: 1 },
  {
    field: 'role',
    headerName: '権限',
    width: 120,
    type: 'singleSelect',
    valueOptions: [
      { value: 'admin', label: '管理者' },
      { value: 'user', label: '一般' },
      { value: 'guest', label: 'ゲスト' },
    ],
  },
  dateColumn({ field: 'createdAt', headerName: '作成日' }),
  actionsColumn({
    onEdit: (row) => navigate(`/users/${row.id}/edit`),
    onDelete: (row) => handleDelete(row.id),
  }),
]);

// 使用例
<K1s0DataTable
  rows={users}
  columns={columns}
  checkboxSelection
  pagination
  pageSize={20}
  toolbar
  exportOptions={{ csv: true }}
  onRowClick={(user) => navigate(`/users/${user.id}`)}
/>
```

### Flutter 版 (`k1s0_ui`)

```dart
K1s0DataTable<User>(
  rows: users,
  columns: [
    K1s0Column(field: 'name', headerName: '氏名', flex: 1, sortable: true),
    K1s0Column(field: 'email', headerName: 'メール', flex: 1),
    K1s0Column(
      field: 'role',
      headerName: '権限',
      type: K1s0ColumnType.singleSelect,
      valueOptions: [
        K1s0ValueOption(label: '管理者', value: 'admin'),
        K1s0ValueOption(label: '一般', value: 'user'),
      ],
    ),
  ],
  getRowId: (user) => user.id,
  checkboxSelection: true,
  onRowTap: (user) => context.go('/users/${user.id}'),
)
```

## 機能 2: Form Generator

### React 版 - MUI コンポーネント + Zod

```tsx
import { createFormFromSchema } from '@k1s0/ui';
import { z } from 'zod';

// Zod スキーマ定義
const userSchema = z.object({
  name: z.string().min(1, '名前は必須です'),
  email: z.string().email('有効なメールアドレスを入力してください'),
  age: z.number().min(0).max(120).optional(),
  role: z.enum(['admin', 'user', 'guest']),
  notifications: z.boolean().default(true),
  bio: z.string().max(500).optional(),
});

// スキーマから MUI フォームを自動生成
const UserForm = createFormFromSchema(userSchema, {
  labels: {
    name: '氏名',
    email: 'メールアドレス',
    age: '年齢',
    role: '権限',
    notifications: '通知を受け取る',
    bio: '自己紹介',
  },
  fieldConfig: {
    role: {
      component: 'Select', // MUI Select
      options: [
        { label: '管理者', value: 'admin' },
        { label: '一般ユーザー', value: 'user' },
        { label: 'ゲスト', value: 'guest' },
      ],
    },
    bio: {
      component: 'TextField',
      multiline: true,
      rows: 4,
    },
    notifications: {
      component: 'Switch', // MUI Switch
    },
  },
  variant: 'outlined', // MUI TextField variant
  columns: 2,
  submitLabel: '保存',
});

// 使用例
<UserForm
  defaultValues={{ role: 'user', notifications: true }}
  onSubmit={handleSubmit}
  onCancel={handleCancel}
/>
```

### Zod 型から MUI コンポーネントへの自動マッピング

| Zod 型 | MUI コンポーネント |
|--------|-------------------|
| `z.string()` | TextField |
| `z.string().email()` | TextField type="email" |
| `z.number()` | TextField type="number" |
| `z.boolean()` | Switch |
| `z.enum([...])` | Select (4項目以下は RadioGroup) |
| `z.date()` | DatePicker |
| `z.array()` | 動的フィールドリスト |

### Flutter 版

```dart
final userFormSchema = K1s0FormSchema<UserInput>(
  fields: [
    K1s0FormFieldSchema(
      name: 'name',
      label: '氏名',
      required: true,
    ),
    K1s0FormFieldSchema(
      name: 'email',
      label: 'メールアドレス',
      required: true,
      type: K1s0FieldType.email,
      validator: EmailValidator().validate,
    ),
    K1s0FormFieldSchema(
      name: 'role',
      label: '権限',
      required: true,
      type: K1s0FieldType.select,
      options: [
        K1s0FieldOption(label: '管理者', value: 'admin'),
        K1s0FieldOption(label: '一般', value: 'user'),
      ],
    ),
    K1s0FormFieldSchema(
      name: 'notifications',
      label: '通知を受け取る',
      type: K1s0FieldType.switchField,
      defaultValue: true,
    ),
  ],
  fromMap: (map) => UserInput.fromMap(map),
  toMap: (user) => user.toMap(),
);

// 使用例
K1s0Form<UserInput>(
  schema: userFormSchema,
  onSubmit: (values) => saveUser(values),
  submitLabel: '保存',
  columns: 2,
)
```
