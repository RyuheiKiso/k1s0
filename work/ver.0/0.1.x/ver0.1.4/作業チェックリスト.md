# k1s0 v0.1.4 作業チェックリスト

## 概要

フロントエンドフレームワーク（React/Flutter）に DataTable コンポーネントと Form Generator を追加する。

- **React**: MUI (Material-UI) コンポーネントをベースに実装
- **Flutter**: Material 3 準拠のカスタム実装

## 現状分析

### 実装済み（@k1s0/ui）
- [x] K1s0ThemeProvider（MUIテーマ管理）
- [x] FormTextField / FormSelect（基本フォーム）
- [x] Toast / ConfirmDialog（フィードバック）
- [x] Loading / EmptyState / ErrorState（状態表示）
- [x] テストインフラ（vitest）

### 実装済み（k1s0_ui）
- [x] Material 3 テーマ
- [x] 基本的なウィジェット構成

### v0.1.4 で実装
- [x] DataTable コンポーネント（React: MUI DataGrid / Flutter: カスタム）
- [x] Form Generator（React: MUI + react-hook-form / Flutter: カスタム）
