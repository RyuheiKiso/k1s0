# k1s0 v0.2.1 作業チェックリスト

## 概要

バックエンドの対応言語に Python（FastAPI）を追加する。

- **`backend-python` テンプレート新設**: feature / domain 両方のスキャフォールド生成
- **CLI 拡張**: `ServiceType` / `DomainType` enum に `BackendPython` 追加
- **フレームワークパッケージ**: Python 版 k1s0 共通ライブラリ（pip）8パッケージ
- **lint ルール Python 対応**: K010/K011/K020/K021/K022 の Python パターン追加
- **CI/CD**: `python.yml` ワークフロー追加

## 受け入れ基準

### 機能要件（テンプレート生成）

- [x] `k1s0 new-feature --type backend-python --name test-service` でサービスが生成される
- [x] 生成物が Python Clean Architecture 構造（pyproject.toml + src レイアウト）に準拠している
- [x] `.k1s0/manifest.json` が生成され、language: "python" が設定されている
- [x] `--with-grpc` で gRPC 関連依存とファイルが追加される
- [x] `--with-db` で SQLAlchemy / Alembic 関連依存が追加される
- [x] `k1s0 new-domain --type backend-python --name test-domain` でドメインが生成される
- [x] `--with-events` でドメインイベント雛形が追加される
- [x] `--with-repository` でリポジトリプロトコルが生成される
- [x] 6言語（Rust/Go/C#/Python/React/Flutter）すべてのテンプレートが正常動作する

### 機能要件（lint）

- [x] K010/K011: Python 必須ディレクトリ・ファイルの不足を検出する
- [x] K020: Python の `os.environ`, `os.getenv` 等を検出する
- [x] K022: Python の `import` 文から CA 違反を検出する

### 機能要件（フレームワーク）
