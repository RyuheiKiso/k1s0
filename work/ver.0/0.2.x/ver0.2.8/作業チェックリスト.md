## 検証中に修正した問題

| 問題 | 修正内容 |
|------|---------|
| Jaeger イメージ名 | `jaegertracing/jaeger` → `jaegertracing/all-in-one` |
| Prometheus 設定 | `storage` / `remote_write` を prometheus.yml から削除し、CLI フラグに移動 |
| OTEL Collector Loki exporter | 無効な `labels` キーを削除 |
| Grafana secrets | `$__file{...}` を dev 用に `admin` 直接指定に変更 |
| Grafana alerting | legacy alerting を無効化（unified_alerting と競合） |
