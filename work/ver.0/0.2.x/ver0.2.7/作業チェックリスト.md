# k1s0 v0.2.7 作業チェックリスト

## 概要

Kotlin エコシステムを包括的にサポート。`backend-kotlin`（Ktor 3.x）と `frontend-android(Kotlin)`（Jetpack Compose）の2テンプレートを新設。テンプレート（feature/domain/screen）、バックエンド framework パッケージ（12個）、Android framework パッケージ（8個）、lint ルール拡張、CI/CD ワークフロー、ドキュメント更新を含む。

## 現状分析

### 実装済み（関連コンポーネント）

- [x] `backend-csharp` テンプレート — JVM に近い構造のリファレンス
- [x] `backend-python` テンプレート — 最新テンプレートのリファレンス
- [x] `frontend-flutter` テンプレート — モバイルフロントエンドのリファレンス（screen テンプレート含む）
- [x] `frontend-react` テンプレート — フロントエンドのリファレンス

### v0.2.7 で実装
