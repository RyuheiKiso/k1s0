# k1s0 v0.2.10 改修方針

## 概要

k1s0 プラットフォーム全体に包括的なバックプレッシャー制御を導入する。新規パッケージ `k1s0-rate-limit` の追加、gRPC ストリーミングフロー制御、Domain Event Bus のバックプレッシャー戦略、キュー深度メトリクスの公開、フロントエンド API クライアントスロットリング、DB コネクションプールのバックプレッシャーを実装する。

#### Rust ミドルウェア統合例

```rust
// tonic (gRPC) ミドルウェア
pub struct RateLimitInterceptor {
    limiter: Arc<dyn RateLimiter>,
}

impl tonic::service::Interceptor for RateLimitInterceptor {
    fn call(&mut self, request: tonic::Request<()>) -> Result<tonic::Request<()>, tonic::Status> {
        match self.limiter.try_acquire() {
            Ok(_permit) => Ok(request),
            Err(RateLimitError::Exceeded { retry_after }) => {
                let mut status = tonic::Status::resource_exhausted("rate limit exceeded");
                status.metadata_mut().insert(
                    "retry-after",
                    retry_after.as_secs().to_string().parse().unwrap(),
                );
                Err(status)
            }
        }
    }
}

// axum (REST) ミドルウェア
pub async fn rate_limit_middleware(
    State(limiter): State<Arc<dyn RateLimiter>>,
    request: axum::extract::Request,
    next: axum::middleware::Next,
) -> axum::response::Response {
    match limiter.try_acquire() {
        Ok(_permit) => next.run(request).await,
        Err(RateLimitError::Exceeded { retry_after }) => {
            axum::response::Response::builder()
                .status(429)
                .header("Retry-After", retry_after.as_secs().to_string())
                .body(axum::body::Body::empty())
                .unwrap()
        }
    }
}
```

### 3. Domain Event Bus バックプレッシャー改善

```rust
// framework/backend/rust/crates/k1s0-domain-event/src/bus/in_memory.rs（変更）

pub struct InMemoryEventBus {
    tx: broadcast::Sender<DomainEvent>,
    strategy: BackpressureStrategy,
    metrics: EventBusMetrics,
    /// ラグ発生時のリカバリ用バッファ
    recovery_buffer: Arc<Mutex<VecDeque<DomainEvent>>>,
    recovery_capacity: usize,
}

impl InMemoryEventBus {
    pub fn new(capacity: usize, strategy: BackpressureStrategy) -> Self {
        let (tx, _) = broadcast::channel(capacity);
        Self {
            tx,
            strategy,
            metrics: EventBusMetrics::new(),
            recovery_buffer: Arc::new(Mutex::new(VecDeque::with_capacity(capacity))),
            recovery_capacity: capacity,
        }
    }

    pub async fn publish(&self, event: DomainEvent) -> Result<(), EventBusError> {
        self.metrics.queue_depth.inc();
        match self.tx.send(event.clone()) {
            Ok(_) => Ok(()),
            Err(_) => match self.strategy {
                BackpressureStrategy::DropOldest => {
                    // リカバリバッファに保存して再試行
                    let mut buf = self.recovery_buffer.lock().await;
                    if buf.len() >= self.recovery_capacity {
                        buf.pop_front(); // 最古を破棄
                        self.metrics.dropped_events.inc();
                    }
                    buf.push_back(event);
                    Ok(())
                }
                BackpressureStrategy::Block => {
                    // 空きができるまで待機（タイムアウト付き）
                    tokio::time::timeout(Duration::from_secs(5), async {
                        loop {
                            if self.tx.send(event.clone()).is_ok() {
                                return Ok(());
                            }
                            tokio::time::sleep(Duration::from_millis(10)).await;
                        }
                    }).await.map_err(|_| EventBusError::Timeout)?
                }
                BackpressureStrategy::Reject => {
                    self.metrics.rejected_events.inc();
                    Err(EventBusError::BufferFull)
                }
                BackpressureStrategy::DropNewest => {
                    self.metrics.dropped_events.inc();
                    Ok(()) // 最新（現在のイベント）を破棄
                }
            }
        }
    }

    /// ラグ発生時にリカバリバッファからイベントを再取得
    pub async fn recover_lagged(&self) -> Vec<DomainEvent> {
        let mut buf = self.recovery_buffer.lock().await;
        buf.drain(..).collect()
    }
}
```

```rust
// framework/backend/rust/crates/k1s0-cache/src/patterns/write_behind.rs（変更）

pub struct WriteBehindQueue<T> {
    tx: mpsc::Sender<T>,
    rx: Arc<Mutex<mpsc::Receiver<T>>>,
    capacity: usize,
    metrics: WriteBehindMetrics,
}

/// Prometheus メトリクス
struct WriteBehindMetrics {
    queue_depth: prometheus::IntGauge,
    queue_capacity: prometheus::IntGauge,
    enqueue_total: prometheus::IntCounter,
    dequeue_total: prometheus::IntCounter,
    rejected_total: prometheus::IntCounter,
}

impl<T: Send + 'static> WriteBehindQueue<T> {
    pub fn new(capacity: usize) -> Self {
        let (tx, rx) = mpsc::channel(capacity);
        let metrics = WriteBehindMetrics::register();
        metrics.queue_capacity.set(capacity as i64);
        Self { tx, rx: Arc::new(Mutex::new(rx)), capacity, metrics }
    }

    pub async fn enqueue(&self, item: T) -> Result<(), BackpressureError> {
        match self.tx.try_send(item) {
            Ok(()) => {
                self.metrics.enqueue_total.inc();
                self.metrics.queue_depth.inc();
                Ok(())
            }
            Err(mpsc::error::TrySendError::Full(_)) => {
                self.metrics.rejected_total.inc();
                Err(BackpressureError::QueueFull)
            }
            Err(mpsc::error::TrySendError::Closed(_)) => {
                Err(BackpressureError::Closed)
            }
        }
    }
}
```

```typescript
// framework/frontend/react/packages/api-client/src/throttle.ts（新規）

export interface ThrottleConfig {
    /** 1秒あたりの最大リクエスト数 */
    maxRequestsPerSecond: number;
    /** 同時接続数の上限 */
    maxConcurrent: number;
    /** キュー上限（超過時は即座にリジェクト） */
    maxQueueSize: number;
}

export class RequestThrottle {
    private queue: Array<() => void> = [];
    private activeCount = 0;
    private tokens: number;
    private readonly config: ThrottleConfig;

    constructor(config: ThrottleConfig) {
        this.config = config;
        this.tokens = config.maxRequestsPerSecond;
        // トークン補充タイマー
        setInterval(() => {
            this.tokens = Math.min(this.tokens + this.config.maxRequestsPerSecond, this.config.maxRequestsPerSecond);
            this.processQueue();
        }, 1000);
    }

    async acquire(): Promise<void> {
        if (this.tokens > 0 && this.activeCount < this.config.maxConcurrent) {
            this.tokens--;
            this.activeCount++;
            return;
        }
        if (this.queue.length >= this.config.maxQueueSize) {
            throw new Error('Request throttle queue full');
        }
        return new Promise<void>((resolve) => {
            this.queue.push(() => { this.activeCount++; resolve(); });
        });
    }

    release(): void {
        this.activeCount--;
        this.processQueue();
    }

    private processQueue(): void {
        while (this.queue.length > 0 && this.tokens > 0 && this.activeCount < this.config.maxConcurrent) {
            this.tokens--;
            const next = this.queue.shift();
            next?.();
        }
    }
}
```

```dart
class RequestThrottle {
  final int maxRequestsPerSecond;
  final int maxConcurrent;
  final int maxQueueSize;
  int _tokens;
  int _activeCount = 0;
  final Queue<Completer<void>> _queue = Queue();

  RequestThrottle({
    required this.maxRequestsPerSecond,
    required this.maxConcurrent,
    this.maxQueueSize = 100,
  }) : _tokens = maxRequestsPerSecond {
    Timer.periodic(const Duration(seconds: 1), (_) {
      _tokens = maxRequestsPerSecond;
      _processQueue();
    });
  }

  Future<void> acquire() async {
    if (_tokens > 0 && _activeCount < maxConcurrent) {
      _tokens--;
      _activeCount++;
      return;
    }
    if (_queue.length >= maxQueueSize) {
      throw Exception('Request throttle queue full');
    }
    final completer = Completer<void>();
    _queue.add(completer);
    return completer.future;
  }

  void release() {
    _activeCount--;
    _processQueue();
  }

  void _processQueue() {
    while (_queue.isNotEmpty && _tokens > 0 && _activeCount < maxConcurrent) {
      _tokens--;
      _activeCount++;
      _queue.removeFirst().complete();
    }
  }
}
```

```kotlin
// framework/frontend/android/packages/k1s0-http/src/.../throttle/RequestThrottle.kt（新規）

class RequestThrottle(
    private val maxRequestsPerSecond: Int,
    private val maxConcurrent: Int,
    private val maxQueueSize: Int = 100,
) {
    private val semaphore = Semaphore(maxConcurrent)
    private val tokenBucket = AtomicInteger(maxRequestsPerSecond)
    private val queue = LinkedBlockingQueue<CompletableDeferred<Unit>>(maxQueueSize)

    init {
        CoroutineScope(Dispatchers.Default).launch {
            while (isActive) {
                delay(1000)
                tokenBucket.set(maxRequestsPerSecond)
                processQueue()
            }
        }
    }

    suspend fun acquire() {
        if (tokenBucket.decrementAndGet() >= 0) {
            semaphore.acquire()
            return
        }
        tokenBucket.incrementAndGet()
        val deferred = CompletableDeferred<Unit>()
        if (!queue.offer(deferred)) {
            throw IllegalStateException("Request throttle queue full")
        }
        deferred.await()
    }

    fun release() {
        semaphore.release()
        processQueue()
    }

    private fun processQueue() {
        while (tokenBucket.get() > 0 && semaphore.availablePermits() > 0) {
            val deferred = queue.poll() ?: break
            tokenBucket.decrementAndGet()
            semaphore.tryAcquire()
            deferred.complete(Unit)
        }
    }
}
```

### 6. DB コネクションプールバックプレッシャー

#### Rust

```rust
// framework/backend/rust/crates/k1s0-db/src/pool.rs（変更）

pub struct PoolConfig {
    pub max_connections: u32,
    /// 待機キューの上限。超過時は即座にエラーを返す
    pub max_waiting: u32,
    /// 待機タイムアウト
    pub acquire_timeout: Duration,
}

pub struct BackpressuredPool {
    inner: PgPool,
    waiting: AtomicU32,
    config: PoolConfig,
    metrics: PoolMetrics,
}

struct PoolMetrics {
    active_connections: prometheus::IntGauge,
    idle_connections: prometheus::IntGauge,
    waiting_count: prometheus::IntGauge,
    rejected_total: prometheus::IntCounter,
    acquire_duration: prometheus::Histogram,
}

impl BackpressuredPool {
    pub async fn acquire(&self) -> Result<PoolConnection<Postgres>, DbError> {
        let current_waiting = self.waiting.fetch_add(1, Ordering::SeqCst);
        if current_waiting >= self.config.max_waiting {
            self.waiting.fetch_sub(1, Ordering::SeqCst);
            self.metrics.rejected_total.inc();
            return Err(DbError::pool_exhausted(
                "connection pool waiting queue full"
            ));
        }
        self.metrics.waiting_count.set(current_waiting as i64 + 1);

        let timer = self.metrics.acquire_duration.start_timer();
        let result = tokio::time::timeout(
            self.config.acquire_timeout,
            self.inner.acquire(),
        ).await;
        timer.observe_duration();

        self.waiting.fetch_sub(1, Ordering::SeqCst);
        self.metrics.waiting_count.dec();

        match result {
            Ok(Ok(conn)) => {
                self.metrics.active_connections.inc();
                Ok(conn)
            }
            Ok(Err(e)) => Err(DbError::from(e)),
            Err(_) => {
                self.metrics.rejected_total.inc();
                Err(DbError::acquire_timeout(self.config.acquire_timeout))
            }
        }
    }
}
```

#### Go

```go
// framework/backend/go/k1s0-db/pool.go（変更）

type PoolConfig struct {
    MaxConnections int           `yaml:"max_connections"`
    MaxWaiting     int           `yaml:"max_waiting"`
    AcquireTimeout time.Duration `yaml:"acquire_timeout"`
}

type BackpressuredPool struct {
    inner      *pgxpool.Pool
    waiting    atomic.Int32
    config     PoolConfig
    metrics    *PoolMetrics
}

func (p *BackpressuredPool) Acquire(ctx context.Context) (*pgxpool.Conn, error) {
    current := p.waiting.Add(1)
    if int(current) > p.config.MaxWaiting {
        p.waiting.Add(-1)
        p.metrics.RejectedTotal.Inc()
        return nil, fmt.Errorf("connection pool waiting queue full (%d/%d)", current, p.config.MaxWaiting)
    }
    p.metrics.WaitingCount.Set(float64(current))
    defer func() {
        p.waiting.Add(-1)
        p.metrics.WaitingCount.Dec()
    }()

    ctx, cancel := context.WithTimeout(ctx, p.config.AcquireTimeout)
    defer cancel()

    timer := prometheus.NewTimer(p.metrics.AcquireDuration)
    defer timer.ObserveDuration()

    conn, err := p.inner.Acquire(ctx)
    if err != nil {
        p.metrics.RejectedTotal.Inc()
        return nil, fmt.Errorf("acquire connection: %w", err)
    }
    p.metrics.ActiveConnections.Inc()
    return conn, nil
}
```

## 注意事項

- **後方互換性**: 既存 API を破壊しない。バックプレッシャー機能は全てオプトインで有効化する
- **k1s0-resilience との棲み分け**: `k1s0-resilience` は「障害への耐性」（サーキットブレーカー、リトライ、タイムアウト）、`k1s0-rate-limit` は「負荷の制御」（レート制限、スロットリング）。セマフォ（同時実行制限）は `k1s0-resilience` に維持する
- **Tier 2 配置**: `k1s0-rate-limit` は Tier 2 とし、Tier 1（`k1s0-error`, `k1s0-config`）のみに依存する。`k1s0-observability`（Tier 2）にも依存するが、Tier 2 同士の依存は許可されている
- **環境変数禁止**: レート制限の設定は全て `config/*.yaml` から読み込む。環境変数による設定は行わない
- **シークレット不要**: レート制限の設定にシークレットは含まれない
- **メトリクス命名**: 全メトリクスに `k1s0_` プレフィックスを付与し、`docs/conventions/observability.md` の命名規約に従う
- **テストカバレッジ**: レート制限アルゴリズムは時間依存のためモック時計を使用したテストを記述する
- **パフォーマンス**: トークンバケットの `try_acquire` は O(1)、スライディングウィンドウは O(n) だがウィンドウサイズが小さいため実用上問題なし。ホットパスでロック競合が発生しないよう `AtomicU64` を活用する
