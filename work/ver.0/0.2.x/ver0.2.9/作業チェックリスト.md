## 備考

### 除外された言語と理由

- **Dart**: tree-sitter-dart に tree-sitter 0.25 (ABI version 15) 互換のクレートが存在しない
- **YAML**: tree-sitter-yaml が tree-sitter 0.20 ABI を使用しており、0.25 と互換性なし
- **Dockerfile**: tree-sitter-dockerfile が tree-sitter 0.20 ABI を使用しており、0.25 と互換性なし

### 高度な機能の実装状況

以下の機能は将来的な拡張として計画されているが、v0.2.9 では未実装:

- re-export (`pub use`) 経由の間接依存追跡（K022）
- 変数経由の SQL 文字列構築の追跡（K050）
- 型エイリアスの1段階解決（K026）
- クロスファイル解析
- 型推論

### tree-sitter バージョンアップ理由

tree-sitter を 0.24 から 0.25 にアップグレードした理由:

- Rust/Go/TypeScript/Python/C# の公式 tree-sitter バインディングが ABI version 15 を要求
- ABI version 15 は tree-sitter 0.25 で導入された
- `streaming-iterator` クレートを追加して新しい `QueryMatches` API に対応
