## 10. 技術スタック

- バックエンド: Go, Rust Edition2024（axum + tokio）
- フロントエンド: Flutter, React
- CLI: Rust（clap）
- データベース: PostgreSQL
- キャッシュ: Redis
- コンテナオーケストレーション: Kubernetes
- モニタリング/ロギング: OpenTelemetry（Prometheus/Grafana などは運用で選定）
- API ドキュメント: OpenAPI（Swagger）, Protocol Buffers（gRPC）
