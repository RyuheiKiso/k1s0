# Contributor Covenant 行動規範

## 私たちの誓い

メンバー、コントリビュータ、リーダーとして、私たちは k1s0 コミュニティへの参加を、
年齢、体格、目に見えるまたは見えない障害、民族性、性別の特徴、ジェンダー・アイデンティティ
および表現、経験のレベル、教育、社会経済的地位、国籍、容姿、人種、カースト、肌の色、宗教、
性的アイデンティティおよび指向にかかわらず、誰にとってもハラスメントのない体験にすることを
誓います。

私たちは、開かれた、温かい、多様で、包括的で、健全なコミュニティに貢献する形で行動し、
やりとりすることを誓います。

## 私たちの基準

私たちのコミュニティに前向きな環境をもたらす行動の例:

- 他者に共感と優しさを示すこと
- 異なる意見、視点、経験を尊重すること
- 建設的なフィードバックを与え、上品に受け取ること
- 私たちの過ちによって影響を受けた人々に責任と謝罪を示し、その経験から学ぶこと
- 個人ではなく、コミュニティ全体にとって最善であることに集中すること

容認されない行動の例:

- 性的な言葉やイメージの使用、あらゆる種類の性的な誘いかけ
- トローリング、侮辱や軽蔑的なコメント、個人的または政治的な攻撃
- 公私にかかわらない嫌がらせ
- 本人の明示的な許可なしに、住所や電子メールアドレス等の他者の個人情報を公開すること
- 仕事の場で合理的に不適切とみなされる他の行為

## 施行責任

コミュニティリーダーは、許容される行動の基準を明確化し施行する責任を負い、
不適切、脅迫的、攻撃的、または有害とみなされる行為に対して、
適切で公正な是正措置を取ります。

コミュニティリーダーは、本行動規範に沿わないコメント、コミット、コード、wiki 編集、
issue、その他の貢献を編集または削除する権利と責任を持ち、適切な場合はモデレーションの
判断理由を伝えます。

## 適用範囲

本行動規範は、すべてのコミュニティの場、および個人がコミュニティを公的に代表している
場合にも適用されます。私たちのコミュニティを代表する例には、公式の電子メールアドレスの
使用、公式のソーシャルメディアアカウントを介した投稿、オンラインまたはオフラインの
イベントでの任命された代表者としての行動が含まれます。

## 施行

虐待的、嫌がらせ、その他の容認されない行動の事例は、リポジトリのメンテナに
報告できます。報告は GitHub の private vulnerability reporting 機能、または
[`SECURITY.md`](SECURITY.md) に記載の経路で行ってください。

すべての苦情は速やかかつ公正に確認・調査されます。

すべてのコミュニティリーダーは、いかなる事件の通報者のプライバシーとセキュリティも
尊重する義務があります。

## 施行ガイドライン

コミュニティリーダーは、本行動規範に違反すると判断された行為に対する処分を決定する際、
これらのコミュニティへの影響に関するガイドラインに従います。

### 1. 訂正

**コミュニティへの影響**: 不適切な言葉の使用、またはプロフェッショナルでないと
みなされる、もしくはコミュニティで歓迎されないと考えられる他の行為。

**結果**: コミュニティリーダーからの非公開で書面による警告。違反に関する明確化と
なぜその行動が不適切だったのかの説明。公開での謝罪が要請される場合があります。

### 2. 警告

**コミュニティへの影響**: 単一の事件または一連の行動による違反。

**結果**: 結果を伴う警告。指定された期間、本行動規範の施行についての一方的な
やりとりを含め、関連する人々とのやりとりはありません。これには、コミュニティの場や
ソーシャルメディアなどの外部チャネルでのやりとりの回避も含まれます。これらの条件に
違反すると、一時的または永久的な禁止につながる可能性があります。

### 3. 一時禁止

**コミュニティへの影響**: 持続的な不適切行為を含む、コミュニティ基準に対する深刻な違反。

**結果**: 指定された期間、コミュニティとのいかなる種類のやりとりや公的な
コミュニケーションからの一時的な禁止。本行動規範の施行を含め、関連する人々との
公的または私的なやりとりは、この期間中認められません。これらの条件に違反すると、
永久的な禁止につながる可能性があります。

### 4. 永久禁止

**コミュニティへの影響**: 持続的な不適切行為、個人への嫌がらせ、または個人の集団に
対する攻撃や軽視を含む、コミュニティ基準に違反するパターンを示すこと。

**結果**: コミュニティ内のあらゆる種類の公的なやりとりからの永久的な禁止。
